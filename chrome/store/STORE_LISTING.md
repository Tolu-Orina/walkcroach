## Detailed description

WalkCroach helps solo operators and small teams work from the page they are already on—job boards, storefronts, listings, and support inboxes—without building automation workflows.

**What it does**
- Summarize the current page and ask follow-up questions
- Draft short replies you can paste yourself
- Save captures into workspaces you control
- Recall what you saved later
- Optional sector quick actions (e.g. candidate summary, price track) on matched sites

**What it does not do**
- It does not scrape the web in the background
- It does not auto-submit forms or run browser automation
- It does not require host access at install—site permission is requested only when you summarize or save
